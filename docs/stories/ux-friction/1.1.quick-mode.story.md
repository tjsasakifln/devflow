# Story 1.1: Modo Rápido — `feature new --quick`

**Epic:** UX Friction Reduction — Devflow Onboarding
**Sprint:** 1 (Curto Prazo)
**Priority:** P0 — Bloqueante
**Effort:** 5 pontos
**Status:** InProgress

## Problem Statement

Usuário leigo leva ~35 minutos do `devflow install` até conseguir gerar o primeiro prompt de implementação. Precisa preencher manualmente 4 artifacts (requirements.md 15 seções, roadmap.md 13 seções, actions.md, test-plan.md) — ~41 seções no total. A maioria abandona antes de escrever a primeira linha de código.

## User Story

AS a lazy developer who just wants to code
I WANT to type `devflow feature new "my idea" --quick`
SO THAT all 4 artifacts are AI-generated from my one-line description in under 2 minutes

## Acceptance Criteria

1. `devflow feature new <name> --quick` accepts a one-line feature description and generates all 4 required artifacts (requirements.md, roadmap.md, actions.md, test-plan.md) using AI
2. Generated artifacts pass DoD check 01 (requirements complete), 02 (design documented), 03 (actions done), and 23 (test plan complete)
3. Each generated artifact includes a `🤖 AI-GENERATED — REVIEW BEFORE CODING` header banner
4. If no AI provider configured, `--quick` prompts to run `devflow ai init` first with clear instructions
5. `--quick` flag is shown in `devflow feature new --help`
6. User receives time-saved summary: "⏱️ 4 artifacts generated in 45s. Manual fill would take ~35min."
7. Generated content quality passes sanity score threshold (see Story 3.1)

### Issues Found

1. **REQ-001 (high):** `--quick` CLI option not registered in `src/cli/index.ts`. Missing `.option("--quick")` call means Commander.js cannot parse the flag. Users cannot invoke `devflow feature new <name> --quick`.
2. **MNT-001 (medium):** AC1 partially blocked by REQ-001. The generation code (`feature.ts` + `generator.ts`) is correct but unreachable through the CLI.
3. **TEST-001 (medium):** No CLI option registration test — test coverage focuses on runtime functions but not Commander.js wiring.
4. **REQ-002 (low):** AC7 depends on Story 3.1 — noted and non-blocking.

### Recommended Fix

In `src/cli/index.ts`, add to the `feature new` command:
```typescript
.option("--quick", "Generate all 4 artifacts using AI from a one-line description")
```
And update the action handler to pass `quick`:
```typescript
quick: rawOptions.quick,
```
